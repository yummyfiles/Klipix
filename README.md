##  About

Klipix is a video editor I'm building because I wanted something that felt modern without being overloaded with features.

The goal isn't to compete with professional editing software.

The goal is to make editing simple.

Import your clips, edit your video, export it, and move on.

---

##  Features

- Timeline-based editing
- Multi-track support
- Fast rendering
- Keyboard shortcuts
- Modern desktop interface
- Open source
- Lightweight design

---


##  Goals

Klipix is being built around a few simple ideas:

- Fast performance
- Clean design
- Easy to use
- Open source
- No unnecessary clutter
